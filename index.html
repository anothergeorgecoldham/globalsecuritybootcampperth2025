About Global Security BootCamp

This community-driven event brings together experts, practitioners, and learners to dive deep into Microsoft technologies such as Identity & Access Management, Endpoint & Threat Protection, Cloud Workload Security, Secure DevOps, AI & ML Security, and Zero Trust architectures.

Our program features expert speakers covering cutting-edge topics including Entra ID, Conditional Access, PIM, Defender XDR, Endpoint Manager, Azure Security Centre, Sentinel, Bicep/ARM guardrails, GitHub Copilot for Security, DevSecOps pipelines, and compliance & governance frameworks.
